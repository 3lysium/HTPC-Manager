Features


Xbmc control

Play/pause your movies and tv shows, control your media library and create music playlist. Almost all functions xbmc provides, you can run from Htpc Manager.

Xbmc features
  • Browse library (movies, tvshow, music)
  • Play item
  • Show music playlist
  • Pause, play, stop, skip item.
  • Select subtitle and audiostream.


Tv show management with Sickbeard

If you use Sickbeard to auto-download your tv shows, you can now do it all from within Htpc Manager. All features like Add new show, search episode, next up and show overview can be handled throug Htpc Manager.

Sickbeard features
  • Tvshow overview
  • Show tvshow details
  • List episodes
  • Next airing episodes
  • Search download for episode
  • Add new show to Sickbeard
  • Sickbeard history


Download using Sabnzbd

Sabnzbd might be the most populair multi-platform software for usenet downloads. It is fully automated so you don't have to look at it. And now with Htpc Manager, all it's great features are easily managable and combined with all your other htpc software.

Because the interface for Htpc Manager is responsive and works on all devices, you are always in control of your downloads. Everywhere.

Sabnzbd features
  • Add new nzb
  • View current downloads
  • Show Sabnzbd history
  • Restart failed downloads
  • Assign downloads to category
  • Limit download speed


Multi-device Responsive interface

Htpc Manager is created using Twitter Bootstrap and is compatible with all sorts of devices. Wether you use Htpc Manager on your desktop, tablet or phone, it always looks good! The interface adjust according to your screen size.

On you desktop, white spacing makes sure the interface looks nice and clean, while on your phone the content adjusts size to fint on the screen without vertical scrolling.

Twitter Bootstrap?

Bootstrap was made to not only look and behave great in the latest desktop browsers (as well as IE7!), but in tablet and smartphone browsers via responsive CSS as well.

Compatible software

Xbmc

XBMC is an award-winning free and open source (GPL) software media player and entertainment hub for digital media. XBMC is available for Linux, OSX, and Windows.

It has been designed to be the perfect companion for your HTPC. Supporting an almost endless range of remote controls, and combined with its beautiful interface and powerful skinning engine, XBMC feels very natural to use from the couch and is the ideal solution for your home theater.

Xbmc.org

Sickbeard

Sick Beard is a PVR for newsgroup users. It watches for new episodes of your favorite shows and when they are posted it downloads them, sorts and renames them, and optionally generates metadata for them.

Sickbeard.com

Sabnzbd

SABnzbd is an Open Source Binary Newsreader written in Python.

SABnzbd makes Usenet as simple and streamlined as possible by automating everything it can. All you have to do is add an .nzb. SABnzbd takes over from there, where it will be automatically downloaded, verified, repaired, extracted and filed away with zero human interaction.

Sabnzbd.org

Couchpotato

An automatic NZB and torrent downloader, just fill in what you want to see and CouchPotato will add it to your "want to watch"-list. Every day it will search through multiple NZBs & Torrents sites, looking for the best possible match. Once a correct release is found, it will send it to Sabnzbd.

Couchpota.to

Newznab

Newznab is a usenet indexing application, that makes building a usenet community easy.

Use your favorite newznab-based nzb indexer like nzbs.org, nzb matrix or even your own private index, to search for downloads from within Htpc Manager.

Newznab.com

Python

Python is a programming language that lets you work more quickly and integrate your systems more effectively. Python runs on Windows, Linux/Unix, Mac OS X, and has been ported to the Java and .NET virtual machines.

At the heart of Htpc Manager lies Python. It is used to serve all the pages to the browser and maintains the Api-connections between all software modules.

Python.org
